Hosting a game-

  To host a game, be sure you have an IP address that all can access.  When the server is open, it will tell you your current IP address.  There is a list of 'lines' that will allow up to 6 people to connect.  When a player connected, there name will be listed in one of the lines.  Once you have all players connected (including yourself) you can now start the game.
